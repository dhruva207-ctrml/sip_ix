# Pull Request Checklist

- [ ] Title follows `type(scope): summary` convention (optional)
- [ ] PR description explains changes and any migrations
- [ ] Run `npm run lint` and fix issues
- [ ] Run `npm run build` successfully
- [ ] Accessibility: interactive elements have aria labels, focusable
- [ ] Responsive: verify main breakpoints (mobile/tablet/desktop)
- [ ] Tests added or updated if applicable
- [ ] Update `CHANGELOG.md` with summary
- [ ] Assign reviewers and link issue
